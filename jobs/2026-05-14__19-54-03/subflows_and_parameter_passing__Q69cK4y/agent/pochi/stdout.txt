### Key design decisions

* **No `@task` decorator used** — both flows are pure `@flow` subflows as required.
* **Logging setup**: A `FileHandler` pointed at `/home/user/project/output.log` is configured at module load time so records survive Prefect's internal stdout capture. A `StreamHandler` is also attached to mirror output to the console.
* **Guaranteed file write**: After the flow returns, `_file_handler.flush()` is called and a final `open(..., "a")` write appends the `Result:` line, ensuring the file is fully written to disk before the process exits.
* The script is executable directly with `python3 main.py`.
